# API Endpoints

## HTML API

### Root

- `GET /`

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Emails

- `GET /api/emails`
- `POST /api/emails`
- `GET /api/emails/:id`
- `PATCH /api/emails/:id`
- `DELETE /api/emails/:id`

### Labels

- `GET /api/labels`
- `POST /api/labels/`
- `PATCH /api/labels/id`
- `DELETE /api/Labels/id`
